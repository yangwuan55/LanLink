### MVVM Pattern

- **View**: presentation/MainActivity.kt
- **ViewModel**: presentation/LanViewModel.kt exposes StateFlows

## Key Implementation Details

### NSD Implementation
- **MulticastLock acquired BEFORE registration/discovery** (NsdAdvertiser.kt:42, NsdDiscoverer.kt:50)
- **resolveService() called BEFORE host/port access** (NsdDiscoverer.kt:62-63, 105-127)
- **Dynamic port (port=0)** used in TcpSocketServer.kt:17

### Coroutines Usage
- All socket I/O uses `Dispatchers.IO`
- `CoroutineScope(Dispatchers.IO + SupervisorJob())` in TcpSocketServer/Client
- No `Thread.sleep` anywhere - uses `delay()` from kotlinx.coroutines

### Protobuf Channel
- Length-prefixed framing: 4-byte length + payload
- Uses `protobuf-javalite` (Lite API)
- MAX_MESSAGE_SIZE = 1MB
